WooCommerce Order Notification for Admin

== Description ==

A Simple WooCommerce Order Notification for Admin with toastr jQuery

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woo-order-notification` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on the JQuery toaster Order Notification to remove the notification in the future
